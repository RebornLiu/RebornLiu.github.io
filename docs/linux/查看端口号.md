### 查看mysql的端口占用

```bash
netstat -anpt | grep mysql |grep LISTEN
```

-a 查看所有

-n 不显示别名，显示数字

-p 显示程序信息 如下图77375是进程id

-t 使用tcp协议

不带n的情况 ip显示的是别名
